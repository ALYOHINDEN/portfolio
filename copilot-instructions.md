# Инструкции для Copilot

- Не изменять профессиональный контент без явного запроса.
- Сохранять русский язык интерфейса.
- Использовать семантический HTML5.
- Не добавлять фреймворки и тяжёлые библиотеки без необходимости.
- Не менять визуальный стиль при рефакторинге без отдельной задачи.
- Фото меняется через поле `photo` в `data/profile.json`.
